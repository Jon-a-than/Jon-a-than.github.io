矩阵分解

lu

三角分解

  • [L, U] = lu(matrix) L:置换下三角矩阵, U: 上三角矩阵

qr

正交分解

  • [Q, R] = qr(matrix) Q:同维上三角矩阵
  • [] = qr(matrix)

奇异值分解

  • A = svd(matrix)
  • [U, S, V] = svd(matrix)

文件I/O

save/load

  • save将所有变量保存至xxx.mat文件中
  • save [fileName] [value1] [value2] [...]将 value变量保存至fileName.mat
  • load将变量从默认文件xxx.mat中读取
  • load [fileName] [value1] [value2] [...]将 value变量从fileName.mat文件读出
