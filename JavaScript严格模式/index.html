🟡JavaScript严格模式

开启严格模式

全局开启

在脚本首行标记'use strict'开启全局严格模式

全局使用严格模式导入没有启用严格模式的库时可能会不兼容

函数单独开启

在函数首行标记'use strict'函数语块开启严格模式

默认的开启

在es6语法中强制使用严格模式

严格模式的优势

  1. 可以对静默错误报错
  2. 优化了代码执行效率
  3. 对未来的保留关键字语法约束

严格模式的特性

严格模式会对以下行为会执行报错

禁止隐式全局变量声明

原先赋值未定义的变量时默认声明全局变量的行为被禁止

禁止删除变量名

禁止同名形参

禁止使用前导0表示八进制

应使用es6语法表示八进制(使用前缀0o或0O)

禁止对对象执行违规操作

禁止对只读属性赋值

禁止删除不可删除属性

禁止拓展不可拓展对象

禁止在块级作用域声明全局函数

在非严格模式下会输出两个1
